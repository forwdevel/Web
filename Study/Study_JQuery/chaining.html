Chaining

Until now we have been writing jQuery statements one at a time (one after the other).

However, there is a technique called chaining,
that allows us to run multiple jQuery commands,
one after the other, one the same element(s).

Tip: This way, browsers do not have to find the same element(s) more than once.

To chain an action, you simply append the action to the previous action.

We could also have added more method calls if needed.

Tip: When chaining, the line if code could become quite long.
However, jQuery is not very strict on the syntax;
you can format it like you want, including line breaks and indentations.
